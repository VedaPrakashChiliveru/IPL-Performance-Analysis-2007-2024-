# 🏏 IPL PERFORMANCE ANALYSIS DASHBOARD

### Power BI Data Analytics

An interactive Power BI dashboard designed to analyze Indian Premier League (IPL) performance across teams, players, and seasons. The project provides comprehensive insights into team success, player contributions, match outcomes, and overall tournament trends using advanced data visualization and DAX analytics.

---

## 📖 PROJECT OVERVIEW

The IPL Performance Analysis Dashboard combines historical IPL data to deliver actionable insights into team and player performance.

The dashboard enables users to:

* Analyze overall IPL statistics
* Compare team performance across seasons
* Evaluate player contributions and consistency
* Identify top-performing teams and players
* Track batting and bowling achievements
* Explore trends through interactive visualizations

By integrating multiple performance metrics into a single analytical model, the dashboard helps uncover patterns that influence success in the IPL.

---

## 🎯 PROJECT OBJECTIVE

**Which teams and players have consistently delivered the best performances in IPL history, and what factors contribute to their success?**

This project investigates team dominance, player excellence, and performance trends through interactive business intelligence reporting.

---

## 🛠 DATA PREPARATION & TRANSFORMATION

Several data preparation steps were performed before building the dashboard:

* Data cleaning and validation
* Handling missing records
* Standardizing team and player names
* Creating calculated measures and KPIs
* Building relationships between tables
* Designing an optimized Power BI data model
* Developing reusable DAX measures for analysis
